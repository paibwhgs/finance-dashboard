
## ⚡ 快速部署流程

### Step 1: 创建 GitHub 仓库 (1 分钟)

1. 访问 https://github.com
2. 点击右上角 "+" → "New repository"
3. 填写：
   - Repository name: `finance-dashboard`
   - Public ✅（必须公开）
4. 点击 "Create repository"

### Step 2: 上传文件 (2 分钟)

1. 在 GitHub 仓库页面，点击 "Add file" → "Upload files"
2. 拖拽这 3 个文件到上传区域：
   - `app_deploy.py`
   - `requirements.txt`
   - `README_GitHub.md`（上传后手动重命名为 `README.md`）
3. 点击 "Commit changes"

### Step 3: 部署到 Streamlit Cloud (2 分钟)

1. 访问 https://share.streamlit.io
2. 点击 **"Deploy an app"**
3. 选择 GitHub 仓库 `finance-dashboard`
4. 配置：
   - Main file path: `app_deploy.py`
   - Branch: `main`
   - Python version: `3.11`
5. 点击 **"Deploy!"**

## 🆘 遇到问题？

### 部署失败
- 检查文件是否全部上传
- 查看部署日志（Streamlit Cloud dashboard）
- 确认 requirements.txt 格式

### 加载慢
- 第一次部署慢是正常的
- 后续访问会快很多
- 可以开启 "Always On"（$5/月）

### 数据获取失败
- Yahoo Finance 可能临时限制
- 稍后重试或更换股票
