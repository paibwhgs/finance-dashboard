# 金融数据可视化看板 - Streamlit Cloud 部署版

## 📦 部署文件说明

本文件夹包含完整部署所需文件，可直接上传到 GitHub 仓库用于 Streamlit Cloud 部署。

## 🚀 部署步骤

### 1. 创建 GitHub 仓库

1. 访问 https://github.com
2. 点击 "New repository"
3. 仓库名：`finance-dashboard`（或你喜欢的名字）
4. 设为 Public（公开）
5. 点击 "Create repository"

### 2. 上传文件

上传以下文件到 GitHub 仓库：
- ✅ `app_deploy.py`（主程序，重命名后的 v4.0）
- ✅ `requirements.txt`（依赖包）
- ✅ `README.md`（说明文档）

**上传方式**：
- 方式 1：GitHub 网页拖拽上传
- 方式 2：使用 Git 命令行
  ```bash
  git clone https://github.com/你的用户名/finance-dashboard.git
  cd finance-dashboard
  # 复制文件到该目录
  git add .
  git commit -m "Initial commit"
  git push origin main
  ```

### 3. 部署到 Streamlit Cloud

1. 访问 https://share.streamlit.io
2. 点击 "Deploy an app"
3. 选择你的 GitHub 仓库 `finance-dashboard`
4. 主文件路径：`app_deploy.py`
5. 点击 "Deploy!"

### 4. 获取分享链接

部署成功后，你会获得类似这样的链接：
```
https://yourusername-finance-dashboard-app-deploy-xyz123.streamlit.app/
```

任何人都可以访问这个链接使用你的应用！

## 🆘 常见问题

### Q: 部署失败怎么办？
检查：
1. 文件是否全部上传
2. `requirements.txt` 格式是否正确
3. GitHub 仓库是否为 Public

### Q: 加载很慢？
- 第一次部署需要 3-5 分钟初始化
- 后续访问会快很多
- 可以在 Streamlit Cloud 设置中开启 "Always On"（需要付费）

### Q: 数据获取失败？
- Yahoo Finance 可能临时限制
- 稍后重试或更换股票
