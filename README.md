# 📈 金融数据可视化看板

深色科技风金融数据 Web 应用，基于 Streamlit + Plotly 构建。

## 功能特性

- 📊 多股票同时查看（AAPL, GOOGL, MSFT 等）
- 📈 交互式 K 线图 + 移动平均线
- 📉 成交量柱状图
- 🔄 实时数据刷新（Yahoo Finance）
- 📥 CSV 数据导出
- 🎨 深色科技风 UI

## 在线访问

**Demo**: [点击访问](https://finance-dashboard-t2qmmfszwbr5ywekfgcllv.streamlit.app/)

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 技术栈

- Streamlit (Web 框架)
- Plotly (图表库)
- yfinance (数据源)

## 数据来源

Yahoo Finance API - 主要支持美股数据
